Présentation du projet

Histoire de rue est un site web permettant de visualiser des rues à partir d'un fichier CSV.
Il s'agit d'un projet open-source (code source -> ici).

Les données

Les données présentées sur la page d'accueil proviennent de OpenData Paris, avec un travail minutieux de labélisation effectué par Sur le Champs.
=> Le fichier source utilisé pour afficher les rues labélisées.

Importer sa carte

Une fonctionnalité d'import est possible. Il vous suffit d'importer un fichier CSV (pint-virgule comme séparateur) ayant les colonnes suivantes :

Nom de la colonne Requis Format de données Commentaires
L_LONGMIN
 Oui Texte Le nom de la rue a afficher
Geometry Oui GeoJson
 Les coordonnées de la rues
display Non Booléen (true/false) Si la rue doit être affichée (en l'absence de cette colonne, toutes les rues sont affichées
category Non Texte La catégorisation de la rue

La cartographie

Le rendu de carte est possible grace aux outils géographiques suivantes :
  • Leaflet est la librairie permettant la navigation sur la carte
  • Jawg Maps Jawg Maps le fournisseur de fonds de carte dynamique et personalisable, basé sur de la donnée Open Street Map
